# Node-RED: 基本的な操作とフローのつくりかた

基本操作とよく使うノードの使い方を学びます

[公式リファレンス](https://nodered.jp/docs/user-guide/nodes)


Node-REDは左から右にノード（機能をもったパーツのようなもの）を繋ぐことでプログラムを作ります。
ノードをつないだものを、フローと呼びます。

各ノードは、次のノードに、Payload（データ、荷物の意味。）を渡します。
